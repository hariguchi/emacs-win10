How to Use Emacs Style Key Binding in Windows 10

Yoichi Hariguchi

Software to Install

  • keyhac to change key binding
  • sharpkeys to swap Caps Lock and Control

Enabling Emacs Style Key Binding

  1. Extract "Keyhac" (say in /cygdrive/c/Program\ Files/keyhac-176)
  2. cd /cygdrive/c/Program\ Files/keyhac-176
  3. mv -i _config.py dot.config.py
  4. wget http://www.hariguchi.org/emacs-win10/config.py Or download config.py and put it in cd /cygdrive/c/Program\ Files/keyhac-176
  5. Run Keyhac.

Swapping Caps Lock and Left Control

This is not about Emacs like key binding itself, but some people (like me) really want to have Control key on the left next to `a'. ShapKeys is useful to swap Caps Lock and Control and here is the procedure.
  1. Run "SharpKeys"
  2. Click "Add"
  3. Choose "Special: Caps Lock (00_3A)" in the "Map this key (From key):" form, and "Special: Left Ctrl(00_1D)" in the "To this key (To key):" form, then click "OK."

  4. Click "Add"
  5. Choose "Special: Left Ctrl(00_1D)" in the "Map this key (From key):" form, and "Special: Caps Lock (00_3A)" in the "To this key (To key):" form, then press "OK."

  6. The result should be like this.

  7. Click "Write to Registry."
  8. Click "Close."
  9. Reboot.
Now Caps Lock and Left Control are supposed to be swapped.
